=============================================
برنامج فحص أرقام الواتساب - Whatsapp Numbers Checker
=============================================

برنامج بسيط لفحص قائمة من الأرقام ومعرفة المسجل منها في واتساب.

طريقة الاستخدام:
----------------
1. قم بشراء مفتاح API من الموقع الرسمي: https://wasenderapi.com
2. تأكد من وجود ملف "api.txt" بجانب البرنامج، وقم بوضع كود الـ API الخاص بك داخله.
3. قم بوضع الأرقام التي تريد فحصها في ملف "numbers.txt" (كل رقم في سطر منفصل).
4. شغل البرنامج "WhatsappChecker.exe".
5. انتظر حتى ينتهي الفحص.

النتائج:
--------
بعد انتهاء البرنامج، ستجد ملفين جديدين:
- valid.txt: يحتوي على الأرقام الصحيحة والتي لديها واتساب.
- invalid.txt: يحتوي على الأرقام التي ليس لديها واتساب.

ملاحظات هامة:
-------------
- يجب توفر اتصال بالإنترنت أثناء عمل البرنامج.
- لا تترك مسافات فارغة أو أسطر إضافية في ملف api.txt.
