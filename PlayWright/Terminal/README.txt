=============================================
برنامج فحص أرقام الواتساب (مجاني - Playwright)
WhatsApp Numbers Checker - FREE
=============================================

برنامج مجاني لفحص أرقام الهاتف ومعرفة المسجل منها في واتساب - بدون API!

طريقة الاستخدام:
================

الخطوة 1: تسجيل الدخول (مرة واحدة فقط)
--------------------------------------
1. شغل البرنامج: python main.py
2. اختر [1] Login to WhatsApp
3. سيفتح المتصفح مع Playwright Inspector
4. امسح رمز QR بتطبيق واتساب على هاتفك
5. بعد تسجيل الدخول، أغلق Inspector
6. تم حفظ الجلسة!

الخطوة 2: فحص الأرقام
---------------------
1. ضع الأرقام في ملف numbers.txt (كل رقم في سطر)
2. شغل البرنامج: python main.py
3. اختر [2] Start Checking Numbers
4. البرنامج سيفحص الأرقام تلقائياً

الملفات:
--------
- numbers.txt: الأرقام المراد فحصها (بالكود الدولي)
- valid.txt: الأرقام الموجودة على واتساب
- invalid.txt: الأرقام غير الموجودة على واتساب
- profiles/: مجلد حفظ جلسة تسجيل الدخول

صيغة الأرقام:
-------------
- الأرقام يجب أن تحتوي على كود الدولة
- مثال: 201012345678 (مصر)
- مثال: 966512345678 (السعودية)

ملاحظات:
--------
- يتطلب اتصال بالإنترنت
- تسجيل الدخول مرة واحدة فقط (الجلسة محفوظة)
- يستخدم خاصية "Add Contact" الرسمية في واتساب
